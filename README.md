# Dacian Rework - Divide et Impera Submod

A visual and stat overhaul for the Dacian faction in Rome 2 Total War, designed for use with the [Divide et Impera](https://divideetimperamod.com/) mod.

---

### ⚔️ What This Mod Does

- Reworks Dacian unit **appearances** for a more authentic look
- Adjusts selected unit **stats** for better faction balance
- Enhances the Dacian roster's visual identity on the battlefield

---

### 📦 Requirements

- **Rome 2 Total War** (latest version)
- **[Divide et Impera (DEI)](https://divideetimperamod.com/)** mod installed and working

---

### 🔧 Installation

1. Download the mod files
2. Place the `.pack` file in your Rome 2 `data` folder:
   ```
   .../Steam/steamapps/common/Total War Rome II/data/
   ```
3. Activate the mod in the Rome 2 launcher **after** DEI in the load order
4. Launch the game and start a new Dacian campaign!

---

### 📝 Notes

- This is a **submod**—DEI must be installed and loaded first
- May not be compatible with other mods that alter Dacian units
- Designed for campaign and custom battles
